# PrevenativeMaintenance
A Display for the Powerapp I built for the PM's we do!

Store Punchlist

Store Punchlist is a maintenance checklist app built to replace the clunky Excel file used for PMs. It gives technicians a cleaner way to work through store devices, record status, add notes, track parts needed, and prepare the punchlist for manager review.
What it helps with

Complete PM checklists by store and device
Mark devices as good, N/A, or needing attention
Capture comments, parts, and supporting photos
Track store health and open maintenance issues
Submit completed punchlists for review
Why it exists

The goal is to make preventive maintenance faster, easier to follow, and less error-prone than managing everything in a spreadsheet.
